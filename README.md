# Blave Quant Skill

A skill that allows an agent to fetch crypto market alpha data from the Blave API.

## Skill

```bash
npx skills add https://github.com/Blave-TW/blave-quant-skill/blave
```

## Setup

No CLI installation required. The blave skill calls the Blave API directly.

Add the following to your `.env`:

```
blave_api_key=YOUR_API_KEY
blave_secret_key=YOUR_SECRET_KEY
```
